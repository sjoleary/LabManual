6.1.1 Species Richness

► Question 6.1

Give a brief definition of species richness.

► Solution

Species richness is a straightforward way to measure the number of species at a given site as the number of species present (more precisely present in the sample).

Let’s calculate species richness across

tidy

richn <- raw_counts %>%
  filter(COUNT > 0) %>%
  group_by(ISLAND, TAXON, HABITAT) %>%
  summarize(n = n(), .groups = "drop")

6.1.2 Species Diversity/Evenness

library(vegan)

diversity <- raw_counts %>%
  filter(COUNT > 1) %>%
  group_by(ISLAND, RUN_TIME) %>%
  summarize(D = n()/sum(COUNT), # counts
            n = n(),            # count adjusted for sample size
            shannon = diversity(COUNT, index = "shannon"),
            simpson = diversity(COUNT, index = "simpson"),
            evenness = diversity(COUNT, index = "simpson")/log(n))            
## `summarise()` regrouping output by 'ISLAND' (override with `.groups` argument)
